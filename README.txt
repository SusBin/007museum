Main purpose
============

The main purpose of this site is to attract visitors to the museum.

Users
=====

As a User I want to know see the location and costs of visiting the museum so that I can plan my visit.

As a user I want to see information about the 007 Museum so that I know what exhibits are there to explore.

Information of the webpage
==========================
The main information includes museum’s name, address, location and opening times as well as selected information on exhibits about exciting, interesting & curious aspects of 007 life, and artifacts to attract prospective visitors

The site’s main purpose is to attract visitors to the museum.
